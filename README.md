# Medicare Plus – Healthcare Management System

## Project Description

**Medicare Plus** is a full-stack web-based Healthcare Management System designed to streamline interactions between patients, doctors, and administrators.

The system provides a centralized platform for managing appointments, medical records, prescriptions, and communication, simulating a real-world healthcare environment.


## Technologies Used

PHP
MySQL
HTML, CSS, JavaScript,
XAMPP / Local Server


## Key Features

### Patient Module

Register and login
Book appointments
View medical records
Communicate with doctors
Rate doctors

### Doctor Module

Manage appointments
View patient details
Add prescriptions
Maintain schedules
Communicate with patients

### Admin Module

Manage doctors and patients
View and control appointments
Handle contact messages
System monitoring dashboard

## How to Run

1. Install XAMPP or any local server
2. Place the project folder inside:

   ```
   htdocs/
   ```
3. Import the database:

   * Open phpMyAdmin
   * Import `database.sql`
4. Update database connection:

   ```
   config/database.php
   ```
5. Start Apache & MySQL
6. Open in browser:

   ```
   http://localhost/medicare_plus
   ```

## Purpose

This project was developed to:

Understand full-stack web development
Implement role-based systems (Admin, Doctor, Patient)
Practice database integration using MySQL
Simulate a real-world healthcare management system
